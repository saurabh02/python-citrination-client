citrination_client.client module¶

class citrination_client.client.CitrinationClient(api_key, site='https://citrination.com')¶

Bases: object

Class for interacting with the Citrination api.

__init__(api_key, site='https://citrination.com')¶

Constructor.

Parameters:
  • api_key – Authentication token.
  • site – Specific site on citrination.com to work with. By default this client interacts with

https://citrination.com. This should point to the full url of the site to access. For example, to access the STEEL site on citrination, use ‘https://STEEL.citrination.com‘.
